Data lagras i molnet (Firebase)

Den här sidan läser data från Firebase Realtime Database – ett moln­baserat system. Det innebär att alla som öppnar sidan ser samma siffror, oavsett dator eller webbläsare. Uppdateringar syns direkt för alla.

Vem kan skriva in siffror?

För att redigera ett värde behöver du logga in med ditt föreningskonto (e-post och lösenord). Klicka på valfri cell – om du inte är inloggad visas ett inloggnings­formulär automatiskt.

Alla kan läsa och se statistiken utan att logga in. Det är bara skrivning som kräver behörighet.

Ny användare – så här skaffar du tillgång

För att få redigeringsbehörighet behöver Kent lägga till dig som användare i systemet. Gör så här:

  1. Kontakta Kent Lundgren och be om att få ett konto.
  2. Kent registrerar din e-postadress i systemet – du behöver inte göra något just nu.
  3. Öppna den här sidan, klicka på valfri cell och välj "Glömt lösenordet? Skicka återställningslänk" i inloggningsrutan. Skriv in din e-postadress och klicka på länken.
  4. Du får ett mail från Firebase (avsändare: noreply@skylt-e0c45.firebaseapp.com). Kolla skräppost om det inte dyker upp inom någon minut. Klicka på länken i mailet och välj ett eget lösenord.
  5. Logga in på den här sidan med din e-post och det nya lösenordet.

När du är inloggad kan du klicka på valfri cell och skriva in ett värde. Ändringen sparas direkt i molnet och syns för alla.

Hur tar jag bort ett felaktigt värde och gör cellen gul igen?

Logga in → klicka på cellen → radera siffran i rutan → tryck Enter. Cellen blir gul igen (data saknas) och ändringen sparas i Firebase.

Vad händer om Firebase inte kan nås?

Sidan faller automatiskt tillbaka på BASE_DATA – den grunddata som är hårdkodad i filen. Inga siffror kan gå förlorade.

🔒 Redigering kräver inloggning. Dina ändringar sparas direkt i Firebase och syns för alla besökare.
