Statistik uppdaterad med 2026-data (jan–maj 2026)

Om detta analysverktyg

Detta verktyg visualiserar medlemsstatistik exporterad från Wondr – ett smart affärssystem primärt för träningsbranschen. Men det fungerar lika bra för en kallbadsförening som Bjerreds Saltsjöbad. På sidan Bli medlem kan du läsa om hur man blir medlem i föreningen.
(Det finns också ett alternativt och skojigt sätt att visa hur man blir medlem – en sida som simulerar en gammal 60-talsdator och visar de fyra stegen för att bli medlem i retrostil!)

Datakälla

All data på denna sida kommer från rapporten "Aktiva medlemmar" som finns i Wondr: Öppna rapporten i Wondr →

I statistiken skiljer vi i huvudsak på enskilt medlemskap och familjemedlemskap. I rapporten "Aktiva medlemmar" återfinns endast en familjemedlem per familj – den som först registrerade sig. Övriga familjemedlemmar registreras genom en separat process: Läs om hur man aktiverar familjemedlemmar →

Genom att exportera rapporten till Excel kan man göra en sammanställning. Se vår Excel-sammanställning (bild) →

Vi fortsätter att analysera och tolka denna rapport, tillsammans med andra rapporter som man kan få från Wondr, för att få djupare insikter om medlemsutvecklingen.

🛠️ Teknisk beskrivning

Använda tekniker

  • HTML5 – Semantisk struktur för tillgänglighet
  • CSS3 – Styling med CSS Custom Properties (variabler), Flexbox och Grid
  • JavaScript (ES6+) – Interaktivitet och datahantering
  • Chart.js – Öppen källkodsbibliotek för responsiva diagram

Funktioner

  • Växling mellan diagramtyper – Stapeldiagram för nyregistreringar, linjediagram med area-fyllning för ackumulerad utveckling
  • Total-linje – I ackumulerat läge visas även summan av alla medlemskap
  • Tidsfiltrering – Data kan filtreras på olika tidsperioder
  • Responsiv design – Anpassad layout för mobil, tablet och desktop med CSS Media Queries
  • Modal-dialog – Visar teknisk information utan sidnavigering

Responsiv design

Sidan använder CSS Custom Properties (variabler) tillsammans med @media queries för att anpassa layouten:

  • Mobil (<480px) – Statistikkort i en kolumn, knappar staplade vertikalt, kompakt diagram
  • Tablet (480-768px) – Statistikkort i två kolumner, mellanstort diagram
  • Desktop (>768px) – Full layout med fyra kolumner för statistik
  • Stor skärm (>1200px) – Extra högt diagram för bättre läsbarhet

Databearbetning

Rådata exporterades från Wondr som Excel-fil (.xlsx). Datan bearbetades med Python och pandas för att gruppera medlemskap per månad och kategori. Den bearbetade datan är sedan inbäddad i JavaScript för snabb rendering utan serveranrop.

Kodstruktur

  • <style> – All CSS i dokumentets head med tydliga sektionskommentarer
  • <script> – JavaScript i slutet av body för optimal laddning
  • CSS-variabler i :root för enkel anpassning av tema och storlekar

Externa beroenden

Endast Chart.js laddas från CDN (Content Delivery Network). Ingen annan extern kod används, vilket minimerar säkerhetsrisker och säkerställer snabb laddningstid.
